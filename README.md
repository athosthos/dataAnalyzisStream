Análise de dados de uma empresa de Stream da Índia

Objetivo: Essa análise busca entender o motivo dos cancenlamentos recorrentes e propor soluções para tratar cancelamentos posteiores

Padrões observados:
- Contratos mensais com altos registro de cancelamento.
- Contratos em que possuem 4 ou mais registros para o call center tem alta tendência de serem cancelados.
- Contratos em que o pagamento é atrasado à mais de 20 dias, possuí alta tendência de cancelado.

Soluções:
- Oferecer descontos para planos trimestrais e anuais, para que tenha mais pessoas trocando de plano mensal para trimestral e anual.
- Quando o cliente ligar 3 vezes para o call center, acender um alerta sobre esse cliente, considerando como um cliente de retenção.
- Quando o cliente atrasar sua fatura por mais de 15 dias, acender o alerta sobre esse cliente.
